# 🛒 英国在线零售数据 EDA 与商业分析

本项目基于 Kaggle 的 **[UK Online Retail](数据集链接)** 数据集，对一家英国在线零售商在 **2010年12月至2011年12月** 间的交易数据进行了全面的探索性数据分析 (EDA) 和商业智能 (BI) 挖掘。

## 📊 项目目标

1.  **用户洞察**：构建用户画像，进行**RFM分层**和**同期群留存分析**，以识别高价值用户和流失风险。
2.  **商品策略**：通过**帕累托分析**和**购物篮关联规则**，优化产品组合、捆绑销售和库存管理。
3.  **市场分析**：分析不同国家的销售表现，识别核心市场和增长潜力市场。
4.  **趋势发现**：探究销售在**月度**和**周内**的周期性规律，为精准营销提供时间坐标。
5.  **预测建模**：基于用户行为特征，构建**流失预警模型**。

## 🛠️ 技术栈

*   **数据处理**: Pandas, NumPy
*   **机器学习**: Scikit-learn
*   **可视化**: Matplotlib, Seaborn

## 📈 核心发现摘要
*   **用户**：平台呈现“橄榄型”分布，高价值用户贡献近半，但首月用户流失率高达60%-70%。
*   **商品**：仅23%的核心商品贡献了80%的销售额，爆款效应显著。
*   **时间**：11月为销售高峰，周四为周内黄金销售日，而周六无交易。
*   **地域**：英国本土是绝对基石，荷兰和澳洲客单价高，是潜在高价值市场。
*   **模型**：流失预警模型AUC高达0.995。
